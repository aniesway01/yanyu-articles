# 人身保險合同糾紛再審審查民事裁定書

## 基本資訊

| 項目 | 內容 |
|------|------|
| **案號** | （2024）陝民申7769號 |
| **案由** | 人身保險合同糾紛 |
| **審理法院** | 陝西省高級人民法院 |
| **審理程序** | 民事再審審查 |
| **裁判日期** | 2025-03-10 |
| **當事人** | 王雪迎等4人 vs 新華人壽保險咸陽支公司 |
| **結果** | ✅ 保戶勝訴 — 駁回保險公司再審申請 |

## 案情摘要

王某投保意外傷害保險。在工作時（挪動電動車過程中）摔倒，送醫後被診斷為急性心肌梗死、心源性休克，最終死亡。

- 死亡醫學證明載明死亡原因為「急性心肌梗死」
- 但已生效的另案判決認定該醫療機構存在重大過錯

保險公司以「疾病猝死非意外傷害」為由拒賠。

## 爭議焦點

**王某的死亡屬於「意外傷害」還是「疾病猝死」？舉證責任如何分配？**

## 法院裁判理由

### 舉證責任分配原則：
1. **被保險人/受益人**：對「保險事故發生」承擔舉證責任
2. **保險人**：根據免責條款拒賠的，對「免責事由的存在」承擔舉證責任

### 本案認定：
- 被申請人已提供證據證明王某在挪車過程中摔倒受傷救治 → 完成初步舉證
- 保險公司主張「疾病猝死」屬於免賠事由 → **保險公司需舉證**
- 死亡醫學證明載明「急性心肌梗死」，但出具該證明的醫療機構已被另案認定存在重大過錯 → **證明力不足**
- 保險公司提交的多名證人證言**不足以證明**王某因自身疾病原因猝死
- 結論：保險公司舉證不足，應承擔保險賠償責任

## 保險公司的8項抗辯（全部被駁回）

| 抗辯 | 法院態度 |
|------|---------|
| 王某死亡無意外因素 | 保險公司需舉證「猝死」免責事由，舉證不足 |
| 另案判決未確認意外傷害 | 另案確認醫療機構有重大過錯，削弱了死亡證明的證明力 |
| 證人證明王某身體不適 | 證人證言不足以證明疾病猝死 |
| 王某有心前區不適病史 | 不足以直接證明死亡原因 |
| 被申請人未申請死因鑑定 | 舉證責任在保險公司 |
| 判決將推翻意外傷害理賠認定 | 不採納 |
| 判決將引起行業混亂 | 不採納 |
| 違背訴源治理要求 | 不採納 |

## 實務啟示

| 要點 | 說明 |
|------|------|
| ✅ **「猝死」免責的舉證責任在保險公司** | 保險公司主張疾病猝死不賠，必須自己舉證，不能要求被保險人證明「不是猝死」 |
| ✅ **有瑕疵的死亡證明證明力不足** | 如果出具死亡證明的醫療機構存在過錯，該證明不能作為拒賠依據 |
| ✅ **「摔倒」可構成意外傷害的初步舉證** | 被保險人只需證明「在某活動中摔倒受傷」即完成初步舉證 |
| ⚠️ 保險公司的證人證言權重有限 | 多名證人僅證明身體不適，不足以排除意外因素 |

## 對保戶諮詢的指導意義

1. **意外險理賠時**：保戶只需證明「意外事故發生」（摔倒、碰撞等），不需要排除所有疾病可能
2. **保險公司以「猝死」拒賠時**：要求保險公司提供充分證據證明死因為疾病
3. **死亡證明有問題時**：如果醫療機構存在過錯，可挑戰死亡證明的證明力
4. **不要輕易同意保險公司的死因鑑定要求**：舉證責任在保險公司，不在保戶
