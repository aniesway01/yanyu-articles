# 人身保險合同糾紛再審民事判決書

## 基本資訊

| 項目 | 內容 |
|------|------|
| **案號** | （2025）蘇民再281號 |
| **案由** | 人身保險合同糾紛 |
| **審理法院** | 江蘇省高級人民法院 |
| **審理程序** | 民事再審 |
| **裁判日期** | 2025-12-30 |
| **當事人** | 顧某甲、宋某 vs 某保險公司（永安財產保險） |
| **結果** | ✅ 保戶勝訴 — 撤銷二審判決，維持一審判決（保險公司賠付765,333.1元） |

## 案情摘要

2020年5月8日，顧某甲通過「水滴保」APP為其子顧某乙（未成年人）投保《少兒醫療險》，年度保險金賠付限額600萬元，年保費641元。

2020年6月6日（等待期內），顧某乙因眼底出血至大豐區人民醫院眼科就診，眼科醫生初步診斷「眼底出血（白血病）」。6月8日至丁醫院（血液科）門診，診斷「血象異常待查」。6月10日入院，確診急性淋巴細胞白血病。

顧某乙輾轉多家醫院治療，包括公立的甲醫院和民營的乙醫院。2021年6月16日身故。

保險公司以兩個理由拒賠：
1. 等待期內出險
2. 在民營醫院（乙醫院）的治療費用不屬於保險責任範圍

## 三個關鍵爭議焦點

### 焦點一：等待期條款 — 保戶勝訴 ✅

| 問題 | 法院認定 |
|------|---------|
| 眼科「初步診斷」= 保險合同的「初次確診」？ | **否。** 眼科醫生無資質診斷白血病，「初步診斷」不等於「初次確診」 |
| 被保險人何時「初次確診」？ | 2020年6月8日經丁醫院血液科檢查後方確診，已過30日等待期 |
| 等待期條款是否為免責條款？ | **是。** 屬格式條款，實質免除保險人責任 |
| 投保須知中擴展的免責範圍有效嗎？ | **無效。** 將免責從「初次確診」擴展到「等待期內出現症狀」，加重投保人責任 |

### 焦點二：民營醫院治療費用（本案核心突破） — 保戶勝訴 ✅

**江蘇省高級人民法院的重大突破性裁判：**

法院認定限定「公立醫院」的條款**無效**，理由：

1. **屬限制被保險人主要權利的格式條款**
   - 醫療保險的被保險人有權自主選擇醫療機構
   - 限定公立醫院限制了被保險人的就醫選擇權

2. **不合理地限制了被保險人主要權利**
   - 「二級以上」（醫院等級）→ 合理 ✅
   - 「普通部」（排除特需/外賓/干部病房）→ 合理 ✅
   - 「公立醫院」（排除所有民營醫院）→ **不合理，條款無效** ❌

3. **理由分析：**
   - 民營醫院中有許多優質醫療機構，尤其專科領域具有技術優勢
   - 排除同等資質民營醫院，與促進民營經濟健康發展原則不符
   - 社會辦醫是醫療服務體系的重要組成部分
   - 保險不應成為限制患者獲取優質醫療資源的枷鎖
   - 保險公司未能提供精算依據證明費率僅基於公立醫院數據測算

4. **本案具體情形：**
   - 被保險人從甲醫院（公立）轉至乙醫院（民營）是甲醫院醫囑引導
   - 乙醫院是甲醫院的「分院」
   - 雙方均認可乙醫院在治療白血病方面較為權威
   - 不應苛求被保險人舍近求遠

### 焦點三：保險金數額

一審認定保險責任範圍內醫療費765,333.1元。再審維持。

## 關鍵法律依據

1. **《民法典》第497條第(2)項** — 格式條款無效情形（不合理免除責任、加重對方責任、限制主要權利）
2. **《保險法》第17條第2款** — 免責條款提示說明義務
3. **《保險法》第30條** — 有利於被保險人的解釋原則
4. **《保險法司法解釋(三)》第20條** — 緊急就醫情形
5. **《中國銀保監會健康保險管理辦法》第26條** — 指定醫療機構約定

## 實務啟示（極重要）

| 要點 | 說明 |
|------|------|
| 🔥 **公立醫院限定條款可能無效** | 高院級別認定：在已限定醫院等級和普通部的情況下，進一步限定公立醫院屬不合理限制，格式條款無效 |
| ⚠️ 等待期「初次確診」≠「初步診斷」 | 非專科醫生的懷疑性判斷不構成「初次確診」，應以專科醫生確診為準 |
| ⚠️ 投保須知不得擴大免責範圍 | 將「初次確診」擴展到「出現症狀」的條款無效 |
| ✅ 醫囑轉院具合理性 | 按醫囑轉至關聯醫院屬合理就醫行為 |
| ✅ 保險應分散風險而非限制就醫 | 法院強調保險應是分散風險的工具，不應成為限制獲取優質醫療資源的枷鎖 |
| 💡 保險公司應承擔精算舉證責任 | 保險公司需舉證說明限定公立醫院的合理性（本案未能舉證） |

## 對保戶諮詢的指導意義

1. **如果保單限定「公立醫院」**：可主張該條款無效，尤其在已有等級限定的情況下
2. **如果在等待期內有「初步診斷」但未「確診」**：應關注是否由專科醫生作出確定性診斷
3. **如果需要轉院至民營醫院**：確保有公立醫院的醫囑或轉院記錄
4. **投保時注意**：互聯網投保的彈窗提示是否真正被閱讀和理解
