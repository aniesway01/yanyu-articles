# 人身保險合同糾紛申請再審民事裁定書

## 基本資訊

| 項目 | 內容 |
|------|------|
| **案號** | （2025）渝民申2048號 |
| **案由** | 人身保險合同糾紛 |
| **審理法院** | 重慶市高級人民法院 |
| **審理程序** | 民事再審審查 |
| **裁判日期** | 2025-06-30 |
| **當事人** | 張某某 vs 某保險公司 |
| **結果** | ❌ 保戶敗訴 — 駁回再審申請 |

## 案情摘要

張某某於2020年11月投保重大疾病保險（2.0版），合同約定「前症疾病」12種，其中第11.1.4條：**多發性大腸腺瘤性息肉（手術）**，需滿足全部條件：
1. 經腸道內鏡及活檢確認為大腸多發息肉
2. 病理學診斷為腺瘤性息肉
3. **進行多發性大腸腺瘤性息肉的腸段切除手術**

張某某被確診為結腸多發息肉，但其所做的是 **CSP術**（冷圈套息肉切除術），而非合同要求的「腸段切除手術」。保險公司拒賠。

## 法院裁判理由

1. 前述條款是**對保險責任範圍的約定**，不屬於限制或免除保險責任的條款
2. 獲得理賠需**同時滿足**全部條件
3. 張某某所做的CSP術 ≠ 腸段切除手術，**不符合合同約定**
4. 保險公司不應承擔給付保險金及豁免保險費的責任

## 張某某的抗辯（全部被駁回）

| 抗辯 | 法院態度 |
|------|---------|
| 拒賠條款無效 | 不採納。屬保險責任範圍約定，非免責條款 |
| PPT宣傳中未提及需腸段切除 | 不採納。以合同條款為準 |
| 醫療機構診斷明確 | 不採納。確診了但手術方式不符合約定 |

## 實務啟示

| 要點 | 說明 |
|------|------|
| ❌ **保險責任範圍條款 ≠ 免責條款** | 法院區分：界定什麼情況才賠（責任範圍）vs 什麼情況不賠（免責），前者不需要特別提示說明 |
| ❌ **手術方式必須完全匹配** | 合同約定「腸段切除」，做了「息肉切除」不算符合 |
| ⚠️ PPT宣傳不能替代合同條款 | 銷售PPT的內容不構成合同約定 |
| 💡 投保前務必看清疾病定義的「全部條件」 | 重大疾病險的理賠通常需同時滿足多個條件 |

## 對保戶諮詢的指導意義

1. **購買重疾險前**：必須仔細閱讀每種疾病的完整定義，特別是手術要求
2. **就醫時**：如果有可能涉及理賠，應與醫生溝通合同對手術方式的要求
3. **保險責任範圍條款 vs 免責條款**：保戶主張「未充分提示說明」的抗辯，僅適用於免責條款，不適用於保險責任範圍條款
