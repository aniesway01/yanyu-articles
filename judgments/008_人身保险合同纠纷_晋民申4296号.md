# 人身保險合同糾紛再審審查民事裁定書

## 基本資訊

| 項目 | 內容 |
|------|------|
| **案號** | （2024）晉民申4296號 |
| **案由** | 人身保險合同糾紛 |
| **審理法院** | 山西省高級人民法院 |
| **審理程序** | 民事再審審查 |
| **裁判日期** | 2024-11-27 |
| **當事人** | 閆某1 vs 某保險公司（太平洋人壽） |
| **結果** | ✅ 保戶勝訴 — 駁回保險公司再審申請 |

## 案情摘要

閆某2（死者，曾為保險公司銷售人員）騎電動車被發現倒地，顱腦外傷致死亡。交警出具的《道路交通事故證明》載明「交通事故成因無法判定」。

保險公司以兩個理由拒賠：
1. 閆某2無證駕駛「超標電動車」（經鑑定屬機動車），屬免責條款約定情形
2. 閆某2曾為保險公司員工，已知免責條款

## 法院裁判理由

**核心認定：死亡原因不明 → 保險公司拒賠理由不成立**

- 交警證明「道路交通事故成因無法判定」
- 閆某2**是否因駕駛電動車造成交通事故導致死亡，原因不明**
- 保險公司稱「無證駕駛機動車導致死亡」→ **缺乏充分證據證實**
- 因此，依據免責條款拒賠**缺乏必要的事實及法律依據**

## 保險公司的抗辯（全部被駁回）

| 抗辯 | 法院態度 |
|------|---------|
| 超標電動車屬機動車，死者無證駕駛 | 事故成因無法判定，無法確認因駕車致死 |
| 已對免責條款盡告知義務 | 拒賠前提（事故成因）不成立，無需審查 |
| 死者曾為保險銷售員，知悉免責條款 | 同上，前提不成立 |

## 實務啟示

| 要點 | 說明 |
|------|------|
| ✅ **事故成因不明時，保險公司拒賠理由不成立** | 保險公司需先證明死亡原因屬於免責情形，成因不明則不能適用免責 |
| ✅ **超標電動車問題不自動導致拒賠** | 即使車輛被認定為機動車，仍需證明是因駕駛該車致死 |
| ✅ **交警無法判定 ≠ 免責事由成立** | 「無法判定」不等於支持保險公司的免責主張 |
| ⚠️ 鑑定意見可被挑戰 | 鑑定材料來源、鑑定方法、鑑定人資質均可質疑 |

## 對保戶諮詢的指導意義

1. **死因不明的意外險理賠**：保險公司必須證明死因屬免責情形，不能僅靠推測
2. **超標電動車事故**：保險公司需先證明事故與駕駛車輛的因果關係
3. **鑑定意見不是終局的**：可從鑑定材料、方法、資質三方面質疑
