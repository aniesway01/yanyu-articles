# 人身保險合同糾紛再審審查民事裁定書

## 基本資訊

| 項目 | 內容 |
|------|------|
| **案號** | （2024）閩民申3655號 |
| **案由** | 人身保險合同糾紛 |
| **審理法院** | 福建省高級人民法院 |
| **審理程序** | 民事再審審查 |
| **裁判日期** | 2024-10-28 |
| **當事人** | 林某某 vs 某甲公司（某乙公司） |
| **結果** | ❌ 保戶敗訴 — 駁回再審申請，保險公司有權解除合同並拒賠 |

## 案情摘要

林某某（女，1983年生）於2021年12月31日通過網絡購買人身保險。投保時健康告知詢問：
- 「近2年是否接受過B超等檢查且結果異常」
- 「是否有腫瘤類疾病（包括結節、囊腫、腫塊等）」

林某某勾選「全部為否」。

**實際情況**：2020年5月6日體檢報告顯示：
- 甲狀腺右側葉不均質回聲區，考慮：結節（內伴鈣化），建議進一步檢查
- 甲狀腺左側葉不均質回聲區，考慮：囊腫
- 雙側乳腺結節
- 膽囊息肉

2022年8月確診**右甲狀腺乳頭狀癌**，申請理賠被拒。

## 林某某的四大抗辯（全部被駁回）

| 抗辯 | 法院態度 |
|------|---------|
| 1. 網上投保僅為流程性勾選，未接受實質詢問 | 網絡投保通過頁面展示詢問內容，投保人需確認「是」或「否」，已構成詢問 |
| 2. 甲狀腺結節≠甲狀腺癌，無直接因果關係 | 法律要求的是「嚴重影響」而非「直接因果關係」；體檢異常→增加理賠風險→足以影響承保決定 |
| 3. 體檢僅為「考慮結節」非確診，屬於「不明知」 | 「異常」≠「確診」；體檢結果異常是客觀事實，未進一步檢查不否認異常事實 |
| 4. 健康告知為概括性詢問，不夠明確具體 | 詢問內容包括B超異常、結節等，已足夠明確具體，非概括性詢問 |

## 法院裁判理由

### 核心認定：投保人違反如實告知義務，保險公司有權解約拒賠

1. **保險公司已盡詢問義務**：健康告知明確詢問了B超檢查異常及結節等症狀
2. **投保人明知體檢異常**：林某某一審中承認曾就體檢結果諮詢過醫生 → 對異常結果「明知」
3. **「未確診」≠「不明知」**：體檢結果顯示異常並被建議進一步檢查 → 存在患病較大可能
4. **影響承保決定**：體檢異常相對於未發現異常，必然增加理賠風險，足以影響保險人承保決定
5. **保險公司解約拒賠合法**：林某某違反法定和約定的如實告知義務，違反最大誠信原則

## 關鍵法律依據

- 《保險法》第16條（如實告知義務）
- 《保險法司法解釋(二)》第5條（如實告知的範圍：投保人明知的內容）
- 《保險法司法解釋(二)》第6條（保險人詢問的範圍和內容為限）

## 實務啟示

| 要點 | 說明 |
|------|------|
| ❌ **體檢異常必須如實告知** | 即使體檢結果用語為「考慮」而非「確診」，仍屬需要告知的「異常」 |
| ❌ **「未確診」≠「不明知」** | 體檢報告載明異常+建議進一步檢查 → 投保人已「明知」異常事實 |
| ❌ **網絡投保的健康告知同樣有效** | 通過頁面展示詢問內容並要求勾選確認，構成有效詢問 |
| ❌ **多項未告知可認定「故意」** | 甲狀腺結節+乳腺結節+膽囊息肉+CIN等多項未告知 → 可能被認定為故意 |
| ⚠️ **「嚴重影響」vs「因果關係」** | 法律要求的是「對保險事故發生有嚴重影響」，不要求直接因果關係 |

## 對保戶諮詢的指導意義

1. **投保前**：所有體檢異常結果（即使未確診）都應如實告知，包括結節、囊腫、息肉等
2. **健康告知**：仔細閱讀每一項詢問內容，體檢報告中「考慮」「建議進一步檢查」等用語均屬異常
3. **多項異常**：如有多項體檢異常未告知，更容易被認定為「故意」而非「重大過失」
4. **網絡投保**：不要因為是線上投保就忽視健康告知，勾選「全部為否」前務必核實
5. **本案教訓**：即使結節最終發展為癌症的因果關係不確定，但只要體檢異常影響承保決定，未告知即可構成違約
