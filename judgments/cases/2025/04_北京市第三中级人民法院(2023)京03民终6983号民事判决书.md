用人单位在法律规定以外创设劳动合同解除条件效力的认定——某房地产公司诉康某劳动争议案
【案件基本信息】
1. 裁判书字号北京市第三中级人民法院(2023)京03民终6983号民事判决书
2. 案由：劳动争议
3. 当事人原告(被上诉人): 某房地产公司
被告( 上诉人) : 康某
【基本案情】
康某于2020年4月10日入职某房地产公司，担任事业部经理，签订期限自2020 年4月10 日至2023 年4月9日的劳动合同，月工资标准为税后400 00
元，自2 0 2 0 年8 月1 日起调整为税后2 0 0 0 0 元，工资支付至2 0 2 0 年1 2 月31日。
康某主张正常出勤至2021 年2 月15 日，2021 年1 月1 日至2021 年2月15日的工资未支付。某房地产公司不认可康某主张，主张2020年12月22日双方
协商一致根据2021年1月项目落地情况，确认工资及劳动关系情况，因2021年1月项目未落地，依据双方约定，双方劳动关系自2020年12月31日终止，


工资支付至当日，2021年1月1日双方转为合作关系，2021年2月5日某房地产公司向康某发送了解除通知。某房地产公司提交了微信聊天记录、通知书以
证明其主张，其中微信聊天记录显示“康总，您好……最后确定的工资结算情况是这样的：2021年1月如果您跟进的项目能落地，就支付您1月工资，如果
项目没落地，工资支付至2020年12月底。离职手续在确定工资结算日期后我们联系您办理，到时候可以转为集团外部合伙人……”康某回复“好的”;通
知书载有“康某先生……现根据公司与您2020年12月22日协商一致达成的相关约定，您在2021年1月仍无项目落地，故双方之间的《劳动合同》已于
2020年12月31日解除，您的工资结算至2020年12月底……”微信聊天记录显示该通知的发送日期为2021年2月8日，康某认可微信聊天记录的真实性，
不认可通知书的真实性，不认可证明目的，认可2021年1月其跟进的项目未落地。康某提起仲裁要求某房地产公司支付其2021年1月1日至2021年2月5日
的工资24597.7元并为其出具解除劳动关系证明，仲裁委对康某的该项请求予以支持，某房地产公司不服仲裁结果，向法院提起诉讼。
【案件焦点】
某房地产公司可否和康某约定工作没有成果就提前解除劳动关系且无须支付康某已付出劳动期间的工资。
【法院裁判要旨】
北京市朝阳区人民法院经审理认为：根据双方微信聊天记录的内容，某房地产公司、康某就2021年1月项目不落地则工
资支付至2020年12月31 日已达成了一致意见。现双方均认可项目1月未落地，康某虽主张其正常工作至2021年2月15日，
但其提交的证据不足以证明其主张，故一审法院对康某要求某房地产公司支付2021年1月1日至2021年2月5日工资的请求
不予支持。


北京市朝阳区人民法院依据《中华人民共和国劳动争议调解仲裁法》第六条之规定，判决如下：


四、劳动合同的解除和终止
一、某房地产公司无须向康某支付2021年1月1日至2021年2月5日工资24597.7元；
二、某房地产公司于本判决生效之日起七日内向康某出具解除劳动关系证明；
三、驳回某房地产公司的其他诉讼请求。
康某不服一审判决，提出上诉。北京市第三中级人民法院经审理认为：《中华人民共和国劳动法》第五十条规定：“工资应当
以货币形式按月支付给劳动者本人。不得克扣或者无故拖欠劳动
者的工资。”《中华人民共和国劳动合同法》第二十六条第一款
规定：“下列劳动合同无效或者部分无效：(一)以欺诈、胁迫
的手段或者乘人之危，使对方在违背真实意思的情况下订立或者变更
劳动合同的；(二)用人单位免除自己的法定责任、排除劳动者权利的；(三)
违反法律、行政法规强制性规定的。”《中华人民共和国劳动合同法实施条例》第十三条规定：“用人单位与劳
动者不得在劳动合同法第四十四条规定的劳动合同终止情形之外
约定其他的劳动合同终止条件。”法院对此认为：综合上述规定
可知，《中华人民共和国劳动合同法》规定的解除劳动合同情形属于封闭
体系，排除了用人单位和劳动者在法定情形之外另行约定终止或者解除劳动合
同条件的权利。对于约定解除劳动合同而言，用人单位和劳动者有权依据《中
华人民共和国劳动合同法》第三十七条规定协商一致解除劳动合同，即在合同
成立并生效后，
在未履行或者未完全履行之前，合同当事人通过协商解除合同，使合同效力归于消灭。但法院需要明确的是，法律并未规定用人单位


和劳动者有权约定劳动合同解除权，即用人单位无权通过提前和
劳动者以条件约定的方式，赋予满足条件时用人单位对劳动者的
劳动合同解除权。劳动法律制度上的这一安排旨在确定劳动基准
、落实解雇保护、维护作为弱势一方的劳动者的合法权益。故，
用人单位和劳动者可以在《中华人民共和国劳动合同法》规定的范围内约定解除劳动合同条件，对于约定的解除劳动合同条件超出
法律规定的情形范围且加重劳动者负担的，应当认定为违反法
律强制性规定而归于无效。本案中，某房地产公司主张2020年12月22日和劳动者约定了如果2021年1


月康某跟进的项目仍无法落地，双方合同于2020年12月31日解除，某房地产公司亦无须支付2020年12月底以后康某的工资。法
院对此认为，该约定并未协商一致解除合同，约定的实质系通过设定工作目标条件的方式变相约定了用人单位单方解除劳动者劳
动合同的条件，并以结果为导向无视劳动者劳动付出免除用人单位支付劳动报酬的义务。法院认为该约定系在法律规定以外创设
了用人单位解除或者终止劳动合同的条件，同时剥夺了劳动者提供劳动后取得劳动报酬的合法权利，系用人单位利用强势地位、
免除自身支付工资的法定责任，排除劳动者合法权利的情形，应属无效。故，某房地产公司无权以此抗辩劳动者的工资支付请求
权。康某的该项上诉理由成立，法院予以支持。
二审中康某提交了微信聊天记录证明其在2021年1月1日至2021年2月5 日正常履职工作，某房地产公司虽主张康某未在该
期间正常提供劳动，但其并未提交证据加以证明，应当承担举证不能的不利后果。双方对于康某在职期间的工资标准未存异议，
结合仲裁裁决作出后康某亦未就仲裁结果提起诉讼，故法院认定某房地产公司应当支付康某2021年1月1日至2021年2月5日工
资24597.7元。

共和国劳动合同法实施条例》第十三条，《中华人民共和国民事诉讼法》第一百七十七条第一款第二项之规定，判决如下：

四、劳动合同的解除和终止
【法官后语】
《中华人民共和国劳动合同法》明确规定了劳动合同解除和终止的条件，
但实践中，部分用人单位为了满足自身的经营管理需求，往往会与劳动者达成法定事由之外的解除条件，就该类解除事由的约定是否有效并无明确的法律
规定。
从《中华人民共和国劳动法》《中华人民共和国劳动合同法》立法变迁以及体系解释上看，我国对劳动关系解除事由实行的是法定化的立法模式，即解
除劳动合同“事由法定，程序严格”。《中华人民共和国劳动合同法实施条例》
第十三条规定：“用人单位与劳动者不得在劳动合同法第四十四条规定的劳动合同终止情形之外约定其他的劳动合同终止条件。”该条虽然仅对于劳动合同
的终止进行了限制性规定，但劳动合同作为一种继续性合同，“解除”和“终止”两者均是面对将来而言，都是劳动关系结束的一种方式，合同解除系属于
合同终止的特殊情形，解除就是提前终止，其后果都是消灭了双方劳动合同的权利义务，如果严格进行文义上的区分，允许用人单位在法定条件以外约定劳
动合同解除的条件，那么用人单位完全可以将约定终止条件转变为约定解除条件，从而导致《中华人民共和国劳动合同法实施条例》第十三条的立法目的落
空。实行法定化立法模式的意义在于平衡用人单位和劳动者之间的利益，维护作为弱势一方的劳动者的合法权益，保持劳动关系的稳定性，若是允许用人单
位与劳动者约定法定情形之外的劳动合同解除条件，则很容易产生用人单位利用优势地位免除自身法定责任，侵害劳动者合法权利的情形，也会导致《中华
人民共和国劳动合同法》第四条关于制定规章制度的民主程序、第三十九条关于用人单位依据劳动者严重违反规章制度的行为解除劳动合同的规定难以落实。
对于实践中用人单位为了满足自身的经营管理需求，与劳动者达成法定事由之外的解除条件效力的判断，司法实践中存在不同的裁判观点。有观点认为，
法律并未禁止约定劳动合同的解除条款，劳动合同作为双方约定权利义务的体现，也是契约自由原则的体现，只要约定内容不违反法律、行政法规的强制性
规定，依法可以认定为有效。虽然劳动法具有社会法的属性，《中华人民共和


国劳动法》和《中华人民共和国劳动合同法》均未规定用人单位可以在劳动合同中与劳动者约定解除条件，但劳动合同也是一种合同。根据《中华人民共和
国民法典》的规定，当事人可以约定一方解除合同的事由。解除合同的事由发生时，解除权人可以解除合同。对于作为特殊法的劳动合同法没有规定的，可
以适用《中华人民共和国民法典》的一般规定，只要该约定的解除条件不违反法律、行政法规、国家政策以及公序良俗。也有观点认为，用人单位与劳动者
达成法定事由之外的解除条件效力属于绝对无效，即从劳动法的社会属性，以及维护劳动力市场整体用工秩序和广大劳动者合法权益的角度考虑，应加强对
劳动关系解除事由的管制，用人单位不能在法定解除事由之外自行约定解除条件，即使该约定有利于员工也应当认定为无效。还有相对无效的观点认为，
一般情况下，用人单位与劳动者在劳动合同法规定的劳动合同解除情形之外约定
劳动合同解除条件无效，但双方约定的解除条件未限制劳动者权益，免除用人单位责任的除外。《中华人民共和国劳动合同法实施条例》第十三条规定的目
的在于避免公司利用强势地位排除员工合法权益、免除自身法定责任，如果双方约定的解除劳动合同条件在客观上并没有违背该条法律规定的立法目的，反
而赋予员工更大的自主选择权，考虑保护员工合法权益与促进用工市场灵活性之间的平衡，应认定该约定有效。
本案二审的裁判结论主要基于相对无效的观点，认为双方约定的解除情形可以是对法定情形的合理延伸，但不能超出该范围，更不能约定排除劳动者合
法权益、免除用人单位自身法定责任的条款。在进一步判断用人单位单方行为解除的合法性时，需将其约定条款置于法定解除规定的框架中，以此认定其解
除行为是否合法。
