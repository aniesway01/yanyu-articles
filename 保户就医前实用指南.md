> **用途**：供顧問在接受保戶諮詢時快速查閱，指導保戶在就醫前、就醫中、理賠時採取正確行動

## 第一章：投保前必讀

### 1.1 健康告知 — 如實填寫是鐵律

**案例教訓**：甘肅保險業務員明知已患CIN2仍全勾「否」→ 合同被認定根本無效（003號案）；福建保戶甲狀腺結節+乳腺結節+膽囊息肉全未告知→ 拒賠（010號案）

**行動清單**：
- [ ] 投保前翻出最近2年所有體檢報告
- [ ] 體檢報告中「考慮」「建議進一步檢查」「異常」字樣 = 必須告知
- [ ] 「未確診」≠「不用告知」— 法院認定體檢異常即為「明知」
- [ ] 多項異常未告知 → 更容易被認定為「故意」→ 保險公司可拒賠且不退保費
- [ ] 如實告知後被拒保 → 考慮「簡易健告醫療險」（門檻較低但注意後端限制）

### 1.2 保障項目 — 逐項確認

**案例教訓**：甘肅保戶的殘疾、骨折保障因「可選項未勾選」被拒賠，即使業務員代操作也無法翻盤（009號案）

**行動清單**：
- [ ] 區分「基本保障」和「可選保障」
- [ ] 可選項目逐一確認是否已勾選
- [ ] 業務員代操作 → 務必逐項核對再簽字
- [ ] 簽字 = 確認所有內容，事後很難推翻

### 1.3 疾病定義 — 看清全部條件

**案例教訓**：重慶保戶合同要求「腸段切除」，實際做了「冷圈套切除」→ 不符合（005號案）

**行動清單**：
- [ ] 重疾險每種疾病通常有2-4個條件，必須「全部同時滿足」
- [ ] 特別注意手術方式的要求（名稱必須完全匹配）
- [ ] 銷售PPT、宣傳資料 ≠ 合同條款，以合同為準

### 1.4 就醫範圍 — 注意醫院限制

**行動清單**：
- [ ] 確認合同是否限定「公立醫院」
- [ ] 如果限定公立 → 記下來，理賠時有翻盤空間（見第四章）
- [ ] 確認醫院等級要求（二級及以上？三級？）
- [ ] 確認是否限定「普通部」（排除特需/VIP/國際部）

## 第二章：就醫行動指南

### 2.1 選擇醫院

| 優先級 | 選擇 | 原因 |
|--------|------|------|
| 首選 | 合同約定範圍內的醫院 | 最安全，無爭議 |
| 次選 | 二級及以上公立醫院普通部 | 絕大多數保單都認可 |
| 需要注意 | 民營醫院 | 如果合同限定公立→理賠有爭議但有勝訴案例 |

**如果需要去民營醫院**：
- [ ] 先在公立醫院就診，取得醫囑/轉院建議
- [ ] 保留公立醫院的轉院記錄
- [ ] 確認民營醫院的等級資質（二級及以上）
- [ ] 了解民營醫院是否為公立醫院的合作/分院

### 2.2 就診科室

**案例教訓**：江蘇案保戶在等待期內由眼科「初步診斷」白血病 → 法院認定眼科無診斷白血病資質 → 不算「初次確診」→ 保戶勝訴（002號案）

**行動清單**：
- [ ] 直接到對應專科看診
- [ ] 「初次確診」以專科醫生的明確診斷為準
- [ ] 如果在非專科被「初步懷疑」→ 盡快到專科確認

### 2.3 手術方式

**案例教訓**：重慶案合同要求「腸段切除」，做了「息肉切除」→ 不賠（005號案）

**行動清單**：
- [ ] 就醫前翻出保險合同的疾病定義
- [ ] 確認合同對手術方式的具體要求
- [ ] 與主治醫生溝通合同要求（但以醫療需要為先）
- [ ] 如果合同要求的手術方式不適合病情 → 保留醫生的醫學意見書面記錄

### 2.4 用藥

**行業資訊**：超適應症用藥（醫生根據最新臨床證據開具但超出說明書適應症）可能被拒賠

**行動清單**：
- [ ] 查詢藥品是否在醫保/商保目錄中（國家醫保局公眾號可查）
- [ ] 優先在院內開藥（院外購藥理賠難度大）
- [ ] 如需使用非目錄藥品 → 提前向保險公司報備
- [ ] 保留所有處方、用藥記錄

### 2.5 保留證據

- [ ] 門診病歷（每次就診）
- [ ] 住院病歷（入院記錄、出院小結、手術記錄）
- [ ] 檢查/化驗報告（特別是確診依據的關鍵報告）
- [ ] 醫囑（特別是轉院醫囑）
- [ ] 費用清單和發票（原件）
- [ ] 病理報告（腫瘤類必備）

## 第三章：理賠策略

### 3.1 理賠前檢查清單

- [ ] 確認保險是否在有效期內
- [ ] 確認是否過了等待期
- [ ] 確認疾病/手術是否符合合同定義的全部條件
- [ ] 確認就醫醫院是否符合合同要求
- [ ] 準備完整的理賠材料

### 3.2 費用補償原則

**案例教訓**：甘肅案確認醫療險屬費用補償型，醫保已報銷部分不重複賠付（009號案）

- 費用補償型醫療險 → 已從醫保獲得的部分不重複賠
- 定額給付型（重疾險等）→ 確診即賠，不問實際花費
- 人身保險 → 可與第三人賠償兼得（《保險法》§46）

## 第四章：拒賠應對方案

### 場景A：以「在民營醫院就醫」拒賠

**勝算：高（多省高院支持保戶）**

應對步驟：
1. 確認合同是否已有等級限定（如「二級及以上」）
2. 如果有 → 主張「進一步限定公立」屬不合理限制，格式條款無效
3. 法律依據：《民法典》§497(2)
4. 參考判例：(2025)蘇民再281、(2025)吉民申1046
5. 如有公立醫院轉院醫囑 → 額外強化合理就醫論據
6. 要求保險公司提供精算依據（費率是否基於公立醫院數據測算）

### 場景B：以「猝死/疾病死亡」拒賠意外險

**勝算：中高（取決於現有證據）**

應對步驟：
1. 保戶只需初步證明「意外事故發生」（摔倒、碰撞等即可）
2. 保險公司主張免責 → 舉證責任在保險公司
3. 不要輕易同意保險公司要求的死因鑑定
4. 如果死亡證明出具機構存在過錯 → 挑戰其證明力
5. 參考判例：(2024)陝民申7769、(2024)晉民申4296

### 場景C：以「不符合疾病定義」拒賠重疾險

**勝算：中（取決於保險公司是否充分說明）**

應對步驟：
1. 要求保險公司證明已對疾病具體定義進行特別說明
2. 互聯網投保僅提供鏈接 → 可主張不構成充分說明
3. 保險公司要求額外醫學鑑定 → 主張應基於合同條款判斷
4. 參考判例：(2024)粵民申17778

### 場景D：以「等待期內出險」拒賠

**勝算：中（取決於「初次確診」的認定）**

應對步驟：
1. 確認是否由專科醫生作出確定性診斷
2. 非專科的「初步懷疑」「考慮」≠「初次確診」
3. 如果投保須知將免責擴展到「出現症狀」→ 主張加重投保人責任，條款無效
4. 參考判例：(2025)蘇民再281

### 場景E：以「未如實告知」拒賠

**勝算：低（如果確實未告知）**

誠實評估：
- 如果確實有體檢異常未告知 → 勝算極低
- 即使過了2年，帶病投保可被認定合同根本不成立
- 可嘗試的角度：
  - 健康問卷是否為概括性詢問（如有具體列舉則此抗辯無效）
  - 保險公司是否完成了回溯錄音錄像
  - 未告知事項與保險事故是否有「嚴重影響」關係

## 附錄：關鍵法條速查

| 場景 | 法條 | 要點 |
|------|------|------|
| 格式條款無效 | 《民法典》§497(2) | 不合理限制主要權利→無效 |
| 免責條款未說明 | 《保險法》§17② | 未提示說明→該條款不生效 |
| 條款有歧義 | 《保險法》§30 | 有利於被保險人解釋 |
| 如實告知 | 《保險法》§16 | 投保人對詢問事項如實告知 |
| 免責認定標準 | 《保險法司法解釋(二)》§9 | 免賠額、比例賠付等均可認定免責 |
